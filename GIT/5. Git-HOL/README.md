# 5. Git-HOL

Hands-on Lab covering remote Git repository synchronization, `git pull`, `git push`, repository cleanup, and branch listing verification.

## Objectives Covered

1. **Verify Clean Working State**:
   - Executed `git status` verifying active branch state.
2. **List Available Branches**:
   - Executed `git branch -a` displaying local and `remotes/origin/main` references.
3. **Pull Remote Repository**:
   - Executed `git pull origin main`.
4. **Push Local Changes to Remote Repository**:
   - Staged and committed `GIT/` directory.
   - Executed `git push origin main`.
5. **Verify Remote Reflection**:
   - Confirmed `Your branch is up to date with 'origin/main'`.
