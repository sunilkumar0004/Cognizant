# 1. spring-rest-handson (spring-learn)

Spring Boot project demonstrating Spring Core XML configurations, IoC Container, Bean Scopes, and Logging.

## Objectives Covered

1. **Hands-on 1**: Create Spring Web Maven project (`com.cognizant:spring-learn`).
2. **Hands-on 2**: Spring Core - Load `SimpleDateFormat` from `date-format.xml` (`displayDate()`).
3. **Hands-on 3**: Spring Core - Incorporate logging in `application.properties` with SLF4J logger.
4. **Hands-on 4**: Spring Core - Load `Country` bean from `country.xml` (`displayCountry()`).
5. **Hands-on 5**: Spring Core - Demonstration of Singleton vs Prototype scopes.
6. **Hands-on 6**: Spring Core - Load list of countries from `country.xml` (`displayCountries()`).

## How to Run

### Run Application:
```powershell
.\mvnw.cmd spring-boot:run
```

### Run Tests:
```powershell
.\mvnw.cmd clean test
```
